# 简易编译器前端

本可执行文件实现了一个简易编译器的前端功能：从源代码生成 **中间代码 (IR)** 与 **目标代码 (Assembly-like)**。

## 使用方法

### 1. 默认测试模式

如果你直接运行可执行文件，而不提供任何输入示例，程序会自动使用**内置的默认测试样例**。
运行后，你会得到以下三个输出文件：

* `test.ir` —— 生成的中间代码
* `test.asm` —— 生成的目标代码
* `test.rs` —— 默认测试样例的源程序

### 2. 自定义测试模式

如果你希望使用自己的测试样例：

1. 将你的测试源文件放在**与可执行文件相同的目录下**。
2. 运行可执行文件时，在命令行参数中指定该文件名：

   ```bash
   ./your_executable mytest.rs
   ```
3. 程序会读取该示例并生成对应的：

   * `test.ir` —— 中间代码
   * `test.asm` —— 目标代码
